git强制提交

提交到本地

git add . # 添加所有的文件到本地仓库”
git commit -m “本次提交注释”

强制推送

git push -u origin main -f # main 分支到远程，覆盖远程的内容（注意：远程原有内容会被本地替换）

网络原因无法连接

git config –global http.proxy 127.0.0.1:<系统代理中的端口号> #修改端口号
