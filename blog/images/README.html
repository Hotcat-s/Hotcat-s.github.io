图片资源目录

📁 存放位置

将所有博客和项目的封面图片放在这个目录下。

📝 使用方法

在 Markdown 文件的 front matter 中引用：

---
title: 我的文章
cover: /images/my-cover.jpg
---

在文章内容中引用：

![图片描述](/images/my-image.jpg)

🎨 推荐的封面图片

为了效果统一，建议：

  • 尺寸: 1200x630 px（或保持 1.9:1 比例）
  • 格式: JPG 或 PNG
  • 大小: 不超过 500KB

📂 目录结构示例

images/
├── vue3-reactivity.jpg          # Vue3 响应式原理封面
├── project-blog.jpg             # 博客系统项目封面
├── project-education.jpg        # 在线教育平台封面
├── project-chatbot.jpg          # AI 客服系统封面
├── project-ecommerce.jpg        # 电商管理系统封面
├── project-iot.jpg              # 物联网平台封面
├── project-blockchain.jpg       # 区块链钱包封面
├── project-video-editor.jpg     # 视频剪辑工具封面
└── default-cover.jpg            # 默认封面

🔗 生成后的路径

Hexo 生成后，这些图片会被复制到：

Hotcat-s.github.io/blog/images/

浏览器访问路径：

http://localhost:8000/Hotcat-s.github.io/blog/images/your-image.jpg

💡 提示

  1. 如果没有图片，可以使用默认封面 default-cover.jpg
  2. 图片文件名建议使用英文和连字符，避免中文和特殊字符
  3. 建议对图片进行压缩优化，提升加载速度
